# pipeline_explicabilidad_medica_CNN
Training, evaluation and explicability pipeline with Grad-CAM for CNN models that classify radiological images, with a focus on the clinical validation of the models' activations.

# Pre requisites for 01_import_dataset
1-Complete verification course from https://physionet.org/content/vindr-cxr/1.0.0/
2-Have a kaggle account
2.1-Apply for this contest https://www.kaggle.com/competitions/vinbigdata-chest-xray-abnormalities-detection
